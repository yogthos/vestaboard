# Vestaboard Retro Terminal

A browser-based split-flap display simulator inspired by the Vestaboard.

Displays a 6x22 split-flap board with realistic flip animations and CRT scanline overlay using [Share Tech Mono](https://fonts.google.com/specimen/Share+Tech+Mono) font.

## Usage

Open `index.html` in a browser. Use the buttons at the bottom or keyboard shortcuts (`1`–`5`) to switch between modules. In Custom mode, type a message and press Enter or click Send.
